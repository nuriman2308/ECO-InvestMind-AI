# ECO InvestMind AI

> AI-powered wealth management assistant combining machine learning and natural language processing for personalized stock recommendations.

## 📖 About

ECO InvestMind AI is a cross-platform mobile application designed to help individual investors make more informed financial decisions through artificial intelligence.

The system combines:

- 📈 BERT-based sentiment analysis of financial news
- 🤖 MLP neural network for stock price forecasting
- 🌱 ESG (Environmental, Social, Governance) analysis
- 💼 Personalized investment recommendations
- 📱 Kotlin Multiplatform mobile architecture

Unlike traditional investment applications, ECO InvestMind AI integrates quantitative market data with qualitative news sentiment to provide context-aware investment recommendations, particularly for users with limited financial literacy.

---

## 🚀 Key Features

- Financial news sentiment analysis using BERT
- Stock price prediction with Multilayer Perceptron (MLP)
- ESG-aware investment analysis
- Personalized stock recommendations
- Portfolio optimization
- Cross-platform mobile application (Android & iOS)
- Clean Architecture implementation

---

## 📊 Results

The proposed hybrid AI architecture achieved high predictive performance on the Kazakhstan stock market.

| Model | Task | Performance |
|--------|------|------------:|
| BERT | Financial news sentiment classification | **up to 99.60% accuracy** |
| MLP | Stock price forecasting | **up to 99.74% accuracy** |

The system was evaluated using historical stock prices, ESG metrics, and financial news collected from KASE, Yahoo Finance, and other public financial sources.
